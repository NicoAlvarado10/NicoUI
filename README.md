# NicoUI | La Mejor Colección de Componentes

Bienvenido a **NicoUI**, una colección de componentes bellamente diseñados que puedes copiar y pegar en tus aplicaciones. Estos componentes son accesibles, personalizables y de código abierto, lo que te permite integrarlos fácilmente en varios frameworks como React, Astro, Angular y Vue.

## Características

- **Diseño Moderno**: Componentes visualmente atractivos que mejoran la experiencia del usuario.
- **Accesibilidad**: Diseñados con principios de accesibilidad en mente.
- **Personalización**: Fácilmente personalizables mediante Tailwind CSS.
- **Compatible con Múltiples Frameworks**: Utiliza en React, Astro, Angular, Vue y más.

## Instalación

Para comenzar a usar NicoUI, asegúrate de tener **Tailwind CSS** instalado en tu proyecto. Si aún no lo has hecho, puedes instalarlo siguiendo la [guía de instalación de Tailwind CSS](https://tailwindcss.com/docs/installation).

### Ejemplo de instalación con npm

```bash
npm install tailwindcss
